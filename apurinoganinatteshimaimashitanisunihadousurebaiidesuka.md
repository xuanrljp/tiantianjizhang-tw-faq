# アプリの言語が韓国語になってしまいました。日本語に戻すにはどうすればいいですか？

アプリの言語を日本語に変更する方法は以下の通りです。

▪️方法1

1. iPhoneの「設定」を開きます。
2.  画面を下にスクロールし、「毎日家計簿」を選択します。

    ![list](.gitbook/assets/app\_lang\_jp1.PNG)&#x20;
3. 「言語」を選択します。
4.  「日本語」を選択します。

    ![list](.gitbook/assets/app\_lang\_jp2.PNG)&#x20;

▪️方法2

1. アプリを開きます。
2. 「設定」を選択します。
3. 「言語」を選択します。
4. 「日本語」を選択します。
